# プライバシー・セキュリティチェッカー

## チェック項目

### 1. 個人情報保護（最重要）
- [ ] 選手の個人情報（名前・年齢・位置情報）がログに出力されていないか
- [ ] 動画ファイルのURLが認証なしでアクセスできる状態になっていないか
- [ ] 未成年者（18歳未満）データの処理前に `consent_flag` を確認しているか
- [ ] APIレスポンスに不要な個人情報フィールドが含まれていないか

### 2. 認証・認可
- [ ] 全ての保護エンドポイントに `Authorization: Bearer` チェックがあるか
- [ ] スカウト向けエンドポイントにロールチェック（`role == "scout"`）があるか
- [ ] 自分以外の選手データを書き換えられる脆弱性がないか（IDOR）

### 3. シークレット管理
- [ ] APIキー・シークレットがコードにハードコードされていないか
- [ ] `.env` ファイルへの直接参照がないか（`os.getenv()` を使用しているか）
- [ ] AWS認証情報がコードに含まれていないか

### 4. 入力バリデーション
- [ ] ファイルアップロードでMIMEタイプチェックがあるか（動画のみ許可）
- [ ] SQLインジェクション対策（ORM使用・パラメータバインド）があるか
- [ ] ファイルサイズ制限があるか（動画 最大 500MB）

## チェック後の報告

問題が1件でも見つかった場合は **必ず修正してからコミットする**。
全て問題なければ「セキュリティチェック OK ✅」と一行で報告する。
