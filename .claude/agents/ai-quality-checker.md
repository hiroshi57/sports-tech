# AI品質チェッカー

## チェック項目

### 1. スコア計算の安全性
- [ ] スコアが 0〜100 の範囲外になる可能性はないか
  - `score = max(0, min(100, calculated_score))` でクランプされているか
- [ ] ゼロ除算の可能性はないか（フレーム数・動画長さが0の場合）
- [ ] None / 空の入力に対するガードはあるか

### 2. 非同期処理の安全性
- [ ] Celery タスクにリトライ設定があるか（`max_retries=3`）
- [ ] タスクのタイムアウト設定があるか
- [ ] 失敗時のステータス更新（`status: "failed"`）が実装されているか

### 3. AI評価の透明性
- [ ] スコアを「確定的評価」として提示していないか
- [ ] 分析結果に信頼度スコア（`confidence`）が含まれるか
- [ ] 「参考スコア」の旨をAPIレスポンスに含めるか

### 4. テスト品質
- [ ] 境界値テスト（スコア=0, 100, -1, 101）があるか
- [ ] 空動画・破損動画のエラーハンドリングテストがあるか

## チェック後の報告

上記項目で問題があれば、修正提案を添えてユーザーに報告すること。
全て問題なければ「AI品質チェック OK ✅」と一行で報告する。
