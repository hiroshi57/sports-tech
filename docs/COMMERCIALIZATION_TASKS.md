## 優先着手順（サマリ）

1. **今すぐ (P1)**: #1–#20 でプロダクトを「売れる最小形」に完成させる
2. **並行して (P2)**: #31–#33（法務）と #51–#53（パイロット獲得）は開発と同時に着手 — リードタイムが長いため
3. **P2 の山場**: #21 スコア精度検証 — 「スコアが信用できるか」が外販の生命線
4. **P3**: 課金基盤 (#45) + 営業資材 (#61–#64) + パイロットの有償化 (#56)
5. **P4**: 拡販・多競技・海外

## リスクメモ

- **最大リスク**: 未成年の身体データ・動画を扱うため、法務 (#31–#40) を後回しにすると販売時に全て止まる
- **差別化**: CUJU 等の競合に対しては「育成×セルフケア×スカウト DB の統合」と国内商習慣対応（請求書払い #47）で勝負
- AI スコアは常に「参考スコア」として提示（`is_reference_score: true`）— 選手選考の唯一根拠にしない旨を規約に明記 (#33)
