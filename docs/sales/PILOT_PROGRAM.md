# パイロットプログラム設計 ＋ 導入事例テンプレート（外販 F#41）

> **作成日**: 2026-07-16
> **目的**: パイロット3クラブを獲得し、数値効果付きの導入事例を作る。
> **狙い**: 事例＝最強の営業資材。「同じような組織で効果が出た」を証明する。

---

## 1. パイロットの狙いと成功条件

- **狙い**: 有償契約の前段で、実データ・実運用の効果を数値で証明する。
- **対象**: 育成年代を持つクラブ/スクール（Jユース、街クラブ、強豪高校）を3件。
  タイプを散らす（都市部/地方、規模大/小）と事例の汎用性が上がる。
- **成功条件（KPI）**:
  1. スカウト/指導者の動画レビュー工数を **30%以上削減**
  2. 分析対象選手数を **2倍以上**に拡大（同工数で）
  3. 継続利用意向（NPS/有償転換意向）を獲得

---

## 2. パイロット進行（8週間モデル）

| 週   | フェーズ      | 実施内容                                             | 成果物           |
| ---- | ------------- | ---------------------------------------------------- | ---------------- |
| W0   | キックオフ    | 目的合意・KPI設定・撮影ガイド共有・アカウント発行    | 合意書/KPIシート |
| W1-2 | 撮影・投入    | 対象選手の動画撮影→アップロード→初回分析             | 初期スコア       |
| W3-5 | 運用          | 週次で分析・検索・比較を実運用。定例で使い方サポート | 週次ログ         |
| W6-7 | 効果測定      | 工数・対象人数・発見事例をヒアリング＋ログで定量化   | 効果測定シート   |
| W8   | 振り返り/提案 | 結果レビュー→有償プラン提案→事例化の許諾取得         | 導入事例ドラフト |

## 5. 獲得アプローチ（3クラブの取り方）

1. 既存の人脈・紹介（温かいリード）を最優先。無償パイロットで心理的ハードルを下げる。
2. タイプ違いで3件（例: Jユース1・街クラブ1・高校強豪1）→ 事例の横展開力を確保。
3. パイロット参加のインセンティブ: 初年度割引・優先サポート・事例化での露出。
