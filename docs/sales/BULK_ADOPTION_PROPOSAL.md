# 協会・連盟・自治体 一括導入提案（外販 F#45）

> **作成日**: 2026-07-16
> **目的**: 協会/連盟/自治体への“面”の導入提案。単クラブ営業より大きな契約を狙う。
> **関連**: [PRICING_AND_UNIT_ECONOMICS.md](../PRICING_AND_UNIT_ECONOMICS.md)（Enterprise） / [BULK 契約は E#40 で機能化予定]

---

## 1. なぜ協会・自治体か

- **面で入る**: 傘下の複数クラブ/学校へ一括導入でき、1商談の規模が大きい。
- **政策と合致**: 自治体のスポーツ振興・才能発掘・部活動地域移行の文脈に乗る。
- **公平性の訴求**: 地方・環境に関わらず“数字で”才能を可視化＝機会の平等。
- **コンプライアンス適合**: 未成年データ保護（同意/削除/保存期間）を実装済み。

---

## 2. 提供価値（提案の柱）

| 相手            | 響く価値                                                         |
| --------------- | ---------------------------------------------------------------- |
| 競技団体/協会   | 傘下選手のタレント発掘・強化指定の客観化・育成の標準化           |
| 自治体          | スポーツ振興施策の可視化・地域の才能発掘・部活動地域移行の受け皿 |
| 学校/教育委員会 | 指導の属人性低減・保護者への説明責任（参考値/根拠開示）          |

---

## 3. 導入モデル（3案）

| モデル              | 内容                                         | 課金             |
| ------------------- | -------------------------------------------- | ---------------- |
| A. 一括ライセンス   | 傘下N団体へ Enterprise を一括提供            | 年間・請求書払い |
| B. 選手数パック     | 対象選手○○名までを定額パック                 | 年間・請求書払い |
| C. 実証事業(補助金) | 単年度の実証（自治体予算/補助金）→本格導入へ | 単年度契約       |

> 価格は個別見積（Enterprise）。請求書払い・年度予算に対応（実装済み billing_type=invoice）。

---

## 4. 提案書テンプレート（章立て）

1. 背景と課題（才能の見逃し・指導の属人性・地域格差）
2. 提案概要（面での導入・才能の定量的可視化）
3. 期待効果（発掘数・工数削減・公平性・説明責任）
4. コンプライアンス（未成年データ保護：同意/削除/保存期間）
5. 導入モデルと費用（A/B/C＋Enterprise見積）
6. 導入スケジュール（実証→本格）
7. 実績・事例（[PILOT_PROGRAM.md](PILOT_PROGRAM.md) の数値）
8. 体制・サポート（優先サポート・撮影ガイド提供）

---

## 5. 想定論点と回答

| 論点                   | 回答                                                                        |
| ---------------------- | --------------------------------------------------------------------------- |
| 予算年度・単年度契約   | 請求書払い・年度単位契約に対応。実証事業（補助金）から入れる。              |
| 未成年データの取り扱い | 保護者同意フロー・削除/開示請求・保存期間自動削除を実装（法務ドラフト有）。 |
| 公平性・説明責任       | スコアは“参考値”。根拠内訳・誤差範囲を開示し、人の評価を置換しない。        |
| 情報セキュリティ       | データ削除/エクスポートAPI実装済み。Pマーク/ISMSは取得計画（A#34）。        |
| 現場の運用負荷         | スマホ撮影＋アップロードのみ。撮影ガイドライン冊子を配布。                  |

---

## 6. 残実装（本提案を強くするために）

- **E#40 代理店/クラブ一括契約プラン**: 傘下団体をまとめる組織アカウント/席管理。
- **A#34 Pマーク/ISMS**: 自治体・教育機関の調達要件を満たす認証。
- これらが揃うと「面」の調達要件に正面から応えられる。
