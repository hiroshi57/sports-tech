# 撮影ガイドライン冊子（外販 F#44）

> **作成日**: 2026-07-16
> **目的**: 誰でも分析に適した動画をスマホで撮れるようにする。
> **重要**: 撮影品質はスコアの信頼性・再現性と原価（再処理率）に直結する。
> **関連**: [SCORE_VALIDATION_PROTOCOL.md](../SCORE_VALIDATION_PROTOCOL.md) / [PRICING_AND_UNIT_ECONOMICS.md](../PRICING_AND_UNIT_ECONOMICS.md)

---

## 1. まず守る5原則（これだけで9割OK）

1. **横向き（ランドスケープ）で撮る** — 縦動画は不可。
2. **選手の全身が常に入る** — 足元から頭まで切れないように引きで撮る。
3. **手ブレを抑える** — 三脚 or 固定。歩き撮り・ズーム多用は避ける。
4. **明るい場所で** — 逆光・夕暮れの暗さを避け、順光で。
5. **1プレー＝1動画、5分以内** — 長尺は分割。目的の動作が中心に写るように。

---

## 2. 推奨撮影スペック

| 項目           | 推奨                        | 補足                           |
| -------------- | --------------------------- | ------------------------------ |
| 向き           | 横（16:9）                  | 縦は解析精度が落ちる           |
| 解像度         | 1080p（720p以上）           | 4Kは容量過大で不要             |
| フレームレート | 30fps（60fpsも可）          | 一定であること（可変は避ける） |
| 長さ           | 1〜5分/本                   | 目的動作を中心に               |
| 距離           | 選手全身が画面の6〜8割      | 近すぎ/遠すぎNG                |
| 角度           | 正面 or 斜め45°、地上〜腰高 | 真上・真横の極端な角度は避ける |

---

## 3. シーン別の撮り方

- **ドリブル/スプリント**: 進行方向に対して斜め前から。パン（横移動追従）は滑らかに。
- **ボールコントロール/リフティング**: 正面・全身。足元が切れないよう少し引く。
- **1対1/守備**: 2人が同時に入る画角で。寄せの間合いが分かる距離。
- **シュート/セットプレー**: 助走〜インパクトが1カットに収まるように。

---

## 4. やってはいけない例（再撮影になる）

| NG                    | なぜダメか                   | 対策               |
| --------------------- | ---------------------------- | ------------------ |
| 縦向き撮影            | 全身が入らず解析不能         | 必ず横向き         |
| 頻繁なズーム/手ブレ   | 姿勢推定がブレて誤差増       | 固定・一定の引き   |
| 逆光・暗所            | 体の輪郭が取れない           | 順光・明るい時間帯 |
| 選手が小さすぎ/切れる | 関節が検出できない           | 全身が6〜8割       |
| 複数プレーの長回し    | 目的動作が埋もれる・容量過大 | 1プレー1動画・短く |

---

## 5. アップロード手順（3ステップ）

1. アプリ/Webで対象選手を選ぶ
2. 撮影した横向き動画を選択してアップロード
3. 分析完了通知を待つ（数分）→ カルテでスコア確認

> うまく解析できない場合は、本ガイドの5原則を再確認して撮り直してください。
> 撮影品質が安定すると、スコアの再現性が上がり、再処理も減ります。
