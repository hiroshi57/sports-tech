## 2. 3分デモ台本（読み上げ用）

> 画面は https://web-kappa-steel-92.vercel.app を操作しながら。所要 約3分。

**[0:00 導入 / 15秒]**
「本日は、育成年代の選手を“数字で”見つけて追えるプラットフォームをご覧いただきます。
操作はスマホ撮影とアップロードだけ。まずスカウト画面からお見せします。」

**[0:15 検索 / 30秒]**
（`/scout/search`）「ポジションや地域、総合スコアで公開選手を横断検索できます。
たとえば『FW・スコア70以上』で絞ると、条件に合う選手が一覧に。
一人ひとりのレビュー動画を探し回る必要はありません。」

**[0:45 選手詳細 / 45秒]**
（選手を開く）「これが選手カルテです。4つの基礎スコアをレーダーで、
14の詳細指標、そして重要なのが“このスコアの根拠”。
どの要素がどれだけ寄与したかを開示しています。
さらに誤差範囲も明示。AIが断定するのではなく、判断材料を誠実に出す設計です。」

**[1:30 成長予測 / 30秒]**
「履歴の傾きと年齢から、12ヶ月後の伸びと“伸びしろ”を予測します。
今の実力だけでなく、将来性で選手を評価できます。」

**[2:00 比較・アラート / 40秒]**
（`/scout/compare`）「気になる選手は2〜4人まで重ねて比較。
（`/scout/saved`）検索条件を保存すれば、新しく条件に合う選手が出たとき“新着”で通知。
毎日探しに行かなくても、才能の方から見つかります。」

**[2:40 クロージング / 20秒]**
「撮影ガイドに沿って撮ってアップするだけ。無料で今日から試せます。
未成年データの同意・削除対応も実装済みです。パイロット導入、ぜひご検討ください。」

## 3. デモ環境の案内

- **URL**: https://web-kappa-steel-92.vercel.app
- **ログイン**: 不要（DEMOモードで組み込みデータを表示）
- **見どころ**: `/scout/search`（横断検索）→ 選手詳細（根拠・誤差・成長予測）→
  `/scout/compare`（比較）→ `/scout/saved`（新着アラート）→ `/pricing`（料金）
- **注意**: DEMOデータは匿名・架空。実データ提示時は別途本番テナントを用意。

- 最後の5秒に CTA（無料トライアル/連絡先）とURLを固定表示。
