> **バージョン**: 1.0.0  
> **作成日**: 2026-06-30  
> **ステータス**: ドラフト

---

## 1. プロジェクト背景・目的

### 解決する課題
従来のスポーツスカウティングは、スカウトが現地に足を運び、主観的な観察と統計データだけで選手評価を行ってきた。これには以下の問題がある：

- **地理的制約**: 才能ある選手でも居住地域によって発掘されない
- **コスト**: スカウト活動に多大な移動コストがかかる
- **主観性**: 評価者によって基準がバラバラ
- **選手側の情報格差**: 自分の強み・弱みを客観的に把握する手段がない

### ソリューション
選手が練習動画をアップロードするだけで、AIが客観的な多角的スコアを算出。スカウト・エージェントは世界中の選手データを検索・比較できる。選手自身もプロ選手との比較で具体的な改善ポイントを把握できる。

---

## 2. ターゲットユーザー

### Primary: 選手（アスリート）
- 年齢: 15歳〜28歳（プロ志望の若手中心）
- 状況: 地域クラブ・高校・大学所属、またはフリー
- ニーズ: 客観的な自己評価、プロへの道筋、練習改善

### Secondary: スカウト・エージェント
- 役割: クラブ・チームのスカウト担当、選手エージェント
- ニーズ: 広域での才能発掘、効率的な選手スクリーニング

### Tertiary: コーチ
- 役割: チーム・個人指導者
- ニーズ: 選手の成長可視化、練習メニュー最適化

## 3. 機能要件

### 3.1 コア機能（Phase 1-2 必須）

#### F-01: 動画アップロード・AI分析
- ユーザーが練習動画（MP4, MOV 最大3分）をアップロード
- AIが以下の指標を自動分析・スコア化（0〜100点）
  - 走力（スプリントスピード、加速度、持久力）
  - ボールコントロール（タッチ精度、ファーストタッチ）
  - ポジショニング（スペース活用、判断速度）
  - 体の使い方（姿勢、重心移動、バランス）
  - 総合評価スコア（各指標の加重平均）
- 分析結果は非同期処理（動画提出後5〜10分で通知）
- 分析ごとに「参考スコア」として明示（確定評価でない旨を表示）

#### F-02: 選手プロフィール・データベース
- 基本情報: 名前、年齢、ポジション、所属、身長・体重
- 評価スコア履歴（時系列グラフ）
- アップロード動画一覧
- 公開/非公開設定（スカウトへの開示コントロール）

#### F-03: 活動記録
- 日々の練習・試合の記録（日時、内容、時間、感想）
- 体調・疲労度の自己評価（1〜5段階）
- カレンダービューでの可視化
- 週・月単位の活動量グラフ

### 3.2 スカウト向け機能（Phase 3）

#### F-04: スカウト検索ダッシュボード
- ポジション・年齢・地域・スコア範囲での選手フィルタリング
- 選手プロフィールカード一覧表示
- お気に入り登録・リスト管理
- 選手へのコンタクトリクエスト送信

#### F-05: 比較分析
- 選手同士のスコア比較（最大4名）
- プロ選手・標準データとの比較ベンチマーク
- 動きの重ね合わせ表示（骨格検出によるオーバーレイ）
- レーダーチャート・棒グラフでの視覚化

### 3.3 育成・パーソナライズ機能（Phase 4）

#### F-06: 練習メニュー作成
- AI推奨: 弱点スコアに基づく改善ドリル提案
- カスタム: コーチ・選手が手動でメニュー作成
- 動画付き解説（YouTube埋め込みまたは独自コンテンツ）
- 難易度・所要時間・必要器具のタグ付け

#### F-07: 練習振り返り
- 練習後のAIフィードバック（動画ベース）
- 前回との比較で改善点を自動抽出
- テキスト・音声でのメモ機能
- コーチからのコメント機能

#### F-08: セルフケア
- **怪我リスクモニタリング**: 活動量・疲労度データから過負荷アラート
- **栄養管理**: 食事記録・カロリー・栄養バランス可視化
- **回復状態**: 睡眠時間・体重・疲労度のトレンド分析
- **成長トラッキング**: 身体データ（身長・体重）と能力スコアの相関可視化
- **推奨アクション**: 「今日は軽めのトレーニングを推奨」等のプッシュ通知

## 4. 非機能要件

| 項目 | 要件 |
|---|---|
| パフォーマンス | 動画アップロード完了後10分以内に分析結果を通知 |
| 可用性 | 99.5%以上のアップタイム |
| セキュリティ | 選手個人情報の暗号化保存・HTTPS必須 |
| プライバシー | 未成年者データは保護者同意フラグ必須 |
| スケーラビリティ | 同時動画分析 50件まで対応（初期） |
| 対応OS | iOS 16以上 / Android 12以上 |
| 多言語 | 日本語（初期）/ 英語（Phase 3以降） |

## 5. スコープ

### ✅ やること
- React Nativeモバイルアプリ（iOS / Android）
- FastAPI バックエンド + AI分析エンジン
- Next.js スカウト向けWebダッシュボード
- 動画ストレージ・CDN
- 選手データベース（スカウト検索機能含む）
- 上記F-01〜F-08の全機能

### ❌ やらないこと（スコープ外）
- リアルタイム動画分析（録画→アップロード形式のみ）
- 選手の給与交渉・契約管理
- クラブ運営システム（練習スケジュール管理等）
- ライブ配信機能
- EC機能（用具販売等）
- 独自SNS機能（フォロー・フィード）

---

## 6. 完了判定

以下をすべて満たした時点で「完成」とする：

1. **F-01〜F-03** が動作し、選手がアカウント作成→動画アップロード→スコア確認できる
2. **F-04〜F-05** が動作し、スカウトが選手を検索・比較できる
3. **F-06〜F-08** が動作し、練習メニュー作成とセルフケアアラートが機能する
4. iOS / Android 実機でクラッシュなく動作確認済み
5. セキュリティレビュー（個人情報保護・認証）通過済み
