# データ保存期間・削除ポリシー（D #35 ドラフト）

> **作成日**: 2026-07-13
> **重要**: 弁護士レビュー前の社内ドラフト。プライバシーポリシー(#31)と整合させること。

## 1. 保存期間の原則

必要な期間だけ保持し、目的達成後は削除する（データ最小化）。

| データ種別           | 既定保存期間                                            | 根拠                                         |
| -------------------- | ------------------------------------------------------- | -------------------------------------------- |
| 動画（映像本体・S3） | 最終利用から **24ヶ月**                                 | 成長比較に必要な期間。以降は分析結果のみ残す |
| AI分析結果（スコア） | アカウント有効中                                        | 推移・比較に必要                             |
| 活動/セルフケア記録  | アカウント有効中                                        | 育成ループに必要                             |
| スカウト閲覧ログ     | **12ヶ月**                                              | 監査・本人開示のため                         |
| アカウント（退会後） | 退会後 **速やかに削除**（バックアップは30日以内に消去） | データ最小化                                 |

## 2. 自動削除の実装方針

- 各動画に `last_accessed_at` を持たせ、24ヶ月経過で S3 オブジェクト＋Videoレコードを削除。
- 日次バッチ（Celery beat）で対象抽出 → `delete_s3_object`（実装済）で削除。
- 削除前に選手へ通知（Notification: 「まもなく動画が保存期間満了」）。
- 分析結果は残し、動画のみ削除するオプションも可（スコア推移は保持）。

## 3. 削除・撤回との連動

- 退会（`DELETE /api/account` 実装済 D#33）で関連データを即時削除。
- 保護者の同意撤回（#32）で未成年データを非公開化＋削除キュー投入。
