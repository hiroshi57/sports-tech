# 大学・研究機関との共同研究 企画（外販 A#6）

> **作成日**: 2026-07-16
> **目的**: AIスコアに学術的裏付けを与え、外販時の信頼性（A系）を担保する。
> **関連**: [SCORE_VALIDATION_PROTOCOL.md](SCORE_VALIDATION_PROTOCOL.md)（検証手順） / A#1 相関検証 / A#5 バイアス監査

---

## 1. なぜ共同研究か

- 外部顧客（協会・自治体・教育機関）は「第三者による裏付け」を重視する。
- 自社検証だけでは「お手盛り」批判を免れない。**独立した学術評価**が説得力になる。
- 研究成果（論文・学会発表）はマーケティング資産にもなる。

---

## 2. 研究テーマ候補

| テーマ                                  | 検証する問い                         | 対応する施策 |
| --------------------------------------- | ------------------------------------ | ------------ |
| AIスコアと専門コーチ評価の相関          | AIは熟練者の評価をどれだけ再現するか | A#1          |
| 撮影条件がスコアに与える影響            | 角度/距離/fps でどれだけブレるか     | A#3          |
| スコアの再現性（テスト-再テスト信頼性） | 同一動画で結果は安定するか           | A#4          |
| 年代・体格によるバイアス                | 特定属性が不当に高/低く出ないか      | A#5          |
| 成長予測モデルの妥当性                  | 予測は実際の成長を当てられるか       | B#20         |

---

## 3. 連携先の候補タイプ

- スポーツ科学・体育学部（動作分析・バイオメカニクス研究室）
- 情報工学（コンピュータビジョン・機械学習研究室）
- 医学・リハビリ（動作計測・傷害予防）

## 4. 提供できるもの / 得たいもの

| 当社が提供                         | 当社が得たいもの                   |
| ---------------------------------- | ---------------------------------- |
| 匿名化した分析データ・API アクセス | 独立した精度・妥当性の評価         |
| 開発リソース・計測環境             | 論文/学会発表による第三者裏付け    |
| 研究費・共同研究契約               | モデル改善への科学的フィードバック |

---

## 5. 進め方（フェーズ）

1. **準備**: 研究テーマ確定・匿名化データセット整備・倫理審査（未成年データは特に慎重に）。
2. **予備検証**: 小規模データで相関・再現性の当たりを取る（[SCORE_VALIDATION_PROTOCOL.md](SCORE_VALIDATION_PROTOCOL.md) の手順）。
3. **本検証**: サンプル数を確保し統計的に評価（決定係数・級内相関 ICC 等）。
4. **公表**: 論文・学会発表・ホワイトペーパー化 → 営業デッキ(F#43)に反映。

---

## 6. 留意点

- **未成年データの倫理**: 保護者同意・匿名化・目的外利用の禁止を厳守（D系と整合）。
- **利益相反の開示**: 当社が関与する研究であることを明示し、独立性を担保する設計にする。
- スコアはあくまで**参考値**。研究は「限界の明示」も含めて誠実に行う。
